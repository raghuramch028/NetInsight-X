Feature Engineering & Mapping Methodology

Benchmark Dataset Mapping: Official UNSW-NB15 dataset rows categorize traffic into Normal and attack vectors. Since standard datasets do not map application categories directly, we engineered traffic groupings for normal flows:

Web Browsing Mapped from HTTP, SSL, DNS services with standard small-to-medium sizes.
Streaming Mapped from high-rate UDP streams with continuous transmission sizes.
File Transfer Mapped from FTP/SFTP services or massive TCP segment transfer sizes.
Potentially Suspicious Mapped from raw attack categories (DoS, Backdoor, Fuzzers, etc.).

Why RBF Kernel: Network traffic features represent a complex overlapping space that is not linearly separable. The Radial Basis Function (RBF) Kernel projects these metrics into an infinite-dimensional Hilbert space to construct optimal non-linear classification hyperplanes.
