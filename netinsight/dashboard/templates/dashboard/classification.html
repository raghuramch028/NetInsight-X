Feature Mapping Methodology

Benchmark Dataset Mapping: Official CICIoT2023 records classify traffic into Normal and attack flows. Since standard security sets do not map application classes directly, we engineered groupings:

Normal Standard background TCP/UDP web browsing, SSH control sessions, and idle telemetry flows.
DoS / DDoS Volumetric flooding attacks designed to exhaust resources (TCP SYN Flood, UDP Flood, etc.).
Brute Force / Recon SSH password guessing, vulnerability scans, and port discovery scans.
Mirai Botnet IoT botnet telnet scanner loops and coordinated IoT-device command floods.

Why RBF Kernel: Network traffic dimensions exhibit highly overlapping parameters. The Radial Basis Function (RBF) Kernel projects these metrics into an infinite-dimensional Hilbert space to construct optimal classification boundaries.
