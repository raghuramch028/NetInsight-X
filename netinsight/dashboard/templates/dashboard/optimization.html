Linear Programming Formulation

We formulate bandwidth allocation as a constrained Linear Program. The objective is to maximize the utility of allocated bandwidths $x_i$ weighted by traffic priorities $c_i$, subject to total link capacity $B$ and Class-of-Service QoS constraints:

Maximize   ∑ c_i * x_i
Subject to:
∑ x_i ≤ B   (Total Link Capacity)
x_i ≥ m_i   (Minimum Bandwidth Requirement per Class i)
x_i ≤ M_i   (Maximum Bandwidth Limit per Class i)

Convexity Justification: The objective function is linear (affine), and all constraints are linear inequalities defining a closed convex polytope. A linear optimization problem is structurally convex, ensuring that any local optimum found by the CVXOPT interior-point solver is a global optimum.
